# v3.8.1 — Structural Geometry Evaluation

## Purpose

Controlled DPS evaluation of the structural geometry core introduced in v3.8.0.

## Scope

- Measurement-only: no decoder modifications.
- Evaluate DPS slopes across four modes:
  - Baseline (flooding, no structural)
  - RPC only (flooding, structural RPC enabled)
  - geom_v1 only (geom_v1 schedule, no structural)
  - RPC + geom_v1 (geom_v1 schedule, structural RPC enabled)
- Activation audit per (mode, distance, p).
- Determinism verification.

## Artifacts

- `bench/dps_v381_eval.py` — DPS evaluation harness
- `tests/test_dps_v381_harness.py` — Harness tests
